🎯 Desafio 3 (Nível Avançado)
Desenvolver um sistema interativo via terminal para gerenciar o acervo e os empréstimos de uma biblioteca comunitária.

---

⚙️ Funcionalidades
🔄 Menu interativo com opções para:
[1] Cadastrar Livro
[2] Listar Acervo
[3] Cadastrar Usuário
[4] Emprestar Livro
[5] Devolver Livro
[6] Pesquisar Livro
[7] Mostrar Histórico
[8] Listar Usuários
[0] Sair

📚 Cadastro e gerenciamento de livros com informações: código, título, autor, status de disponibilidade e usuário que está com o livro.

👤 Cadastro e controle de usuários com dados pessoais (nome, CPF, telefone, email, endereço).

🔐 Login administrativo para acesso ao sistema.

✅ Validações para impedir empréstimos de livros já emprestados.

📝 Histórico de movimentações (empréstimos e devoluções).

💬 Feedbacks claros para o usuário em cada ação.
